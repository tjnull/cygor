About Cygor Parse & Quick Guide

Cygor Parse processes scan results from Nmap, Masscan, and other scanning tools to extract and organize host and service information. It automatically generates categorized hostlists for further enumeration.

Key Features:
  • Multi-Format Support: Processes .xml, .nmap, and .gnmap files from various scanners
  • Automatic Categorization: Organizes discovered services into service-specific hostlists
  • Flexible Input: Browse server files, upload files, or extract from zip archives
  • Organized Output: Creates structured directories with parsed results for easy analysis
Input Options Guide:
  • Single File: Parse a single scan file already on the server
  • Directory: Recursively parse all scan files in a directory
  • Upload Files: Upload scan files or zip archives to parse
