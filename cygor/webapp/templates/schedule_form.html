About Scheduler & Quick Guide

Cygor's Task Scheduler allows you to automate recurring security scans and reconnaissance tasks. Schedule port scans, credential testing, and more to run automatically.

Schedule Types
  • Interval: Run every X hours/minutes
  • Cron: Run at specific times (daily at 2 AM)
  • One-time: Run once at a scheduled date
Resource Monitoring

Optionally check CPU and memory usage before running. Tasks will queue until resources are available.

Concurrent Runs

By default, new runs wait for previous ones to complete. Enable concurrent mode to allow parallel execution.

Schedule Types Explained
Interval

Runs the task repeatedly at fixed time intervals. Example: every 6 hours, every 30 minutes.

Best for: Regular recurring scans, monitoring tasks
Cron

Runs at specific times using cron-style scheduling. Example: daily at 2:00 AM, weekdays at noon.

Best for: Maintenance windows, business-hour scans
One-time

Runs exactly once at a specified date and time. Automatically becomes inactive after execution.

Best for: One-off assessments, scheduled pentests
Available Task Types
Port Scan

Network discovery and port enumeration using Masscan, Naabu, and Nmap.

Credential Recon

Test default and common credentials against network services.

Module Scan

Run custom modules like Lockon (screenshots), SMB/NFS explorers.

IOC Enrichment

Enrich indicators with threat intelligence from multiple sources.

Output & Results
