About Task Manager

The Task Manager is your central hub for managing and monitoring all security assessment operations in Cygor. It provides real-time visibility into running tasks, queued operations, and completed assessments.

How It Works
1
Create a Task

Click "New Task" to start a port scan, credential test, or other operation. Configure targets and options.

2
Task Execution

Tasks run in the background. Monitor progress in real-time through the task detail view or check status here.

3
Results Processing

When tasks complete, results are automatically parsed and stored in the database for analysis.

4
View & Export

Access findings through Hosts, Services, and Vulnerabilities views. Export reports for documentation.
