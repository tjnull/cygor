About Scheduled Tasks

Schedule recurring security assessments to run automatically. Set up port scans, credential tests, and more to run at specific times or intervals.

How It Works
1
Create Schedule

Click "New Schedule" to configure a task with targets, options, and timing (cron, interval, or one-time).

2
Automatic Execution

Tasks run automatically at scheduled times. The scheduler handles job queuing and execution.

3
Monitor & Control

View run history, pause/resume schedules, or trigger immediate execution from this dashboard.

4
Review Results

Access results through the Task Manager or dedicated dashboards (CredRecon, etc.).
