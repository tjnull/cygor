About SMBExplorer

SMBExplorer enumerates Windows and Samba file shares across discovered hosts. It attempts authentication (guest, username/password, or NTLM hash), lists available shares, determines access permissions, and optionally enumerates files within accessible shares.

These results help identify misconfigured or sensitive shares that may expose data to unauthorized users.
