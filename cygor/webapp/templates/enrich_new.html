About IOC Enrichment & Quick Guide

IOC Enrichment uses passive reconnaissance and threat intelligence APIs to gather information about IPs, domains, and file hashes. This helps identify malicious infrastructure, track threat actors, and understand attack patterns without active scanning.

Supported Data Sources:
  • Shodan: Discover exposed services, vulnerabilities, and infrastructure details
  • VirusTotal: Check malware detection scores, relationships, and community comments
  • AbuseIPDB: View abuse reports, confidence scores, and threat categories
  • AlienVault OTX: Access threat intelligence pulses and related indicators
  • URLScan.io: Analyze domain reputation and screenshots
Supported IOC Types:
  • IP Addresses: IPv4 addresses (e.g., 192.168.1.1)
  • Domains: Domain names (e.g., example.com)
  • File Hashes: MD5, SHA1, or SHA256 hashes
Supports IP addresses, domains, and file hashes (MD5, SHA1, SHA256). Defanged IOCs will be automatically refanged.
