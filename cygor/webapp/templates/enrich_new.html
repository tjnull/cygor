About IOC Enrichment & Quick Guide

IOC Enrichment uses passive reconnaissance and threat intelligence APIs to gather information about IPs, domains, and file hashes. This helps identify malicious infrastructure, track threat actors, and understand attack patterns without active scanning.

Supported IOC Types:
  • IP Addresses: IPv4 addresses (e.g., 192.168.1.1, 8.8.8.8)
  • Domains: Domain names (e.g., example.com, malware-c2.net)
  • File Hashes: MD5, SHA1, or SHA256 hashes
  • URLs: Full URLs for reputation checking
Output Formats:
  • JSON: Structured data with all enrichment details
  • CSV: Spreadsheet-compatible tabular format
  • XML: Hierarchical markup format
  • Text: Human-readable summary

How IOC Enrichment Works
  1. IOC Parsing

    Your input is parsed and IOC types are automatically detected (IP, domain, hash, URL). Defanged IOCs are refanged.

  2. Source Querying

    Each IOC is queried against selected threat intelligence sources in parallel. Rate limits are respected.

  3. Data Aggregation

    Results from all sources are combined, deduplicated, and enriched with metadata.

  4. Output Generation

    Final results are formatted according to your selection (JSON, CSV, XML, or Text) and saved.

Command Line Options
Input Options
-i, --iocs IOCs to enrich (comma or space separated)
-f, --file Path to file containing IOCs (one per line)
--stdin Read IOCs from standard input
Source Selection
--sources Comma-separated list of sources to query
--all-sources Query all available sources (default)
--exclude Sources to exclude from queries
Output Options
--format Output format: json, csv, xml, text
-o, --output Output file path
--pretty Pretty-print JSON output
Pentester Features
--extract-subdomains Extract subdomains from Wayback/CommonCrawl
--spray-lists Generate credential spray lists from Dehashed
Advanced
--timeout API request timeout in seconds (default: 30)
--retries Max retry attempts per source (default: 3)
Example Commands
Enrich a single IP address
cygor enrich 8.8.8.8
Enrich domain with subdomain extraction
cygor enrich target.com --extract-subdomains --sources wayback,commoncrawl
Generate spray lists from breach data
cygor enrich target.com --sources dehashed --spray-lists
Enrich from file with specific sources
cygor enrich -f iocs.txt --sources shodan,virustotal,abuseipdb
Full pentest enrichment with all features
cygor enrich target.com --extract-subdomains --spray-lists
Custom timeout and retry settings
cygor enrich -f targets.txt --timeout 60 --retries 5
